How to Use Mueblecito

Mueblecito is a tool for designing modular furniture and generating precise cut plans. This guide explains every step and input in detail.

Step-by-Step Guide

  1. Create a Project:
    Click + New Project at the top. Enter a descriptive name (e.g., "Kitchen Cabinets 2026").
  2. Add Modules:
    Click Add Module to add a cabinet, drawer, or custom module. Each module represents a piece of your furniture (e.g., a lower cabinet, an upper cabinet, a drawer unit).
  3. Configure Module Variables:
    For each module, fill in the following fields:
    • Width (ancho):
      Module width in millimeters.
      Example: 600 (for a 60cm wide cabinet)
    • Height (alto):
      Module height in millimeters.
      Example: 720 (for a standard kitchen base cabinet)
    • Depth (profundidad):
      Module depth in millimeters.
      Example: 560 (for a deep cabinet)
    • Material Thickness (espesor):
      Board thickness in mm.
      Example: 18 (for 18mm MDF)
    • Type (tipo):
      Select the module type: Lower Cabinet, Upper Cabinet, Drawer, etc. This affects the parts generated.
    • Back Panel (fondo):
      Choose if the module has a back panel.
      Example: Yes (for closed cabinets), No (for open shelves)
    • Number of Drawers (cajones):
      For drawer modules, set how many drawers you want.
      Example: 3
    • Countertop (cubierta):
      For lower cabinets, select if a countertop is included.
      Example: Yes (for kitchen base units)
    • Custom Name (nombre):
      Optional. Give the module a unique name for easy identification.
  4. Advanced Project Settings:
    In Edit Settings (gear icon or "Edit Settings" button), you can adjust:
    • Default Countertop Thickness: Used for all new modules unless overridden.
    • Default Leg Height: Height of legs for lower cabinets.
    • Panel Discard Margin: Waste margin around each panel (mm).
    • Kerf Width: Saw blade thickness (mm) for accurate cuts.
    • Drawer Slide Clearance: Space per side for drawer slides (mm).
  5. Generate Results:
    Click Generate to see:
    • Panel Summary: List of all required panels by thickness and size.
    • Parts List: Table of all parts, grouped by module.
    • Cutting Diagrams: Visual layout for each board thickness.
    • Pricing: Estimated cost based on your input prices (if available).
  6. Export & Save:
    Download the results as CSV for spreadsheets, or use the diagrams for cutting guidance. Projects are saved in your browser for future editing.

Tips & Best Practices

  • Use clear names for projects and modules to stay organized.
  • Double-check all measurements before generating results.
  • Use the Projects tab to switch between different furniture jobs.
  • Review the Results tab for a summary of all parts and costs.
  • Export your results for sharing or printing.
