Cómo usar Mueblecito

Mueblecito es una herramienta para diseñar muebles modulares y generar planos de corte precisos. Esta guía explica cada paso y campo en detalle.

Guía paso a paso

  1. Crea un proyecto:
    Haz clic en + Nuevo Proyecto arriba. Escribe un nombre descriptivo (ejemplo: "Cocina 2026").
  2. Agrega módulos:
    Haz clic en Agregar Módulo para añadir un gabinete, cajonera u otro módulo. Cada módulo representa una pieza de tu mueble (ejemplo: gabinete bajo, alto, cajonera).
  3. Configura las variables del módulo:
    Para cada módulo, completa los siguientes campos:
    • Ancho (ancho):
      Ancho del módulo en milímetros.
      Ejemplo: 600 (para un gabinete de 60cm)
    • Alto (alto):
      Alto del módulo en milímetros.
      Ejemplo: 720 (gabinete bajo estándar)
    • Profundidad (profundidad):
      Profundidad del módulo en milímetros.
      Ejemplo: 560
    • Espesor (espesor):
      Espesor del material en mm.
      Ejemplo: 18 (para MDF de 18mm)
    • Tipo (tipo):
      Selecciona el tipo de módulo: Gabinete bajo, Gabinete alto, Cajonera, etc. Esto afecta las piezas generadas.
    • Fondo (fondo):
      Indica si el módulo lleva fondo.
      Ejemplo: Sí (para gabinetes cerrados), No (para estantes abiertos)
    • Cajones (cajones):
      Para cajoneras, indica cuántos cajones tendrá.
      Ejemplo: 3
    • Cubierta (cubierta):
      Para gabinetes bajos, selecciona si lleva cubierta.
      Ejemplo: Sí (para muebles de cocina)
    • Nombre personalizado (nombre):
      Opcional. Da un nombre único al módulo para identificarlo fácilmente.
  4. Configuración avanzada del proyecto:
    En Editar configuración (icono de engranaje o botón "Editar configuración"), puedes ajustar:
    • Espesor de cubierta por defecto: Se usa para nuevos módulos si no se especifica otro.
    • Altura de patas por defecto: Altura de las patas para gabinetes bajos.
    • Margen de descarte de panel: Margen de desperdicio alrededor de cada panel (mm).
    • Ancho de corte (kerf): Espesor de la hoja de sierra (mm) para cortes precisos.
    • Juego de corredera de cajón: Espacio por lado para correderas (mm).
  5. Genera resultados:
    Haz clic en Generar para ver:
    • Resumen de paneles: Lista de todos los paneles requeridos por espesor y tamaño.
    • Lista de piezas: Tabla de todas las piezas, agrupadas por módulo.
    • Diagramas de corte: Distribución visual para cada espesor de placa.
    • Precios: Costo estimado según tus precios (si los ingresaste).
  6. Exporta y guarda:
    Descarga los resultados como CSV para hojas de cálculo, o usa los diagramas para el corte. Los proyectos se guardan en tu navegador para futuras ediciones.

Consejos y buenas prácticas

  • Usa nombres claros para proyectos y módulos para mantener el orden.
  • Verifica todas las medidas antes de generar resultados.
  • Utiliza la pestaña Proyectos para alternar entre diferentes trabajos.
  • Revisa la pestaña Resultados para ver el resumen de piezas y costos.
  • Exporta tus resultados para compartir o imprimir.
